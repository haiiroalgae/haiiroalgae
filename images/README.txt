# 图片存放说明

此文件夹用于存放所有静态图片。

## 需要放入的图片

### 必需图片：
- `top.jpg` - 首页顶部16:9大图
- `avatar.png` - 皮套页面立绘图片

### 二创图集 (fanart):
- `fanart1.jpg` ~ `fanart6.jpg`

### 委托展示:
- `commission1.jpg` ~ `commission6.jpg`

### 博客配图:
- `post1.jpg` ~ (根据博客内容添加)

### 角色缩略图和详情图:
- `thumb_a.jpg` ~ `thumb_j.jpg` - 角色小图标
- `detail_a.jpg` ~ `detail_j.jpg` - 角色大图

## 图片尺寸建议

- 首页置顶图: 1920x1080 或等比例
- 立绘: 800x1200 或等比例
- 角色缩略图: 200x200 正方形
- 角色详情图: 800x1000 或等比例
- 博客图片: 800x600 或等比例

请根据实际需求替换图片文件。
